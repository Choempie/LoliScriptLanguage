## LoliScript Language ##
**Loliscript** is a Custom written language for **[Black Bullet](https://bullet.black/)** made by **Ruri.**


This Extenstion adds Snippets and Syntax highlighting.

### The file Extenstion is .loli ###

* FUNCTION
* REQUEST
* PARSE
* SET
* GIB
* KEYCHAIN
* AND MUCH MORE!
